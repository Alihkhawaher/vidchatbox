Step 3: Create Home Screen Icon

  1. Long press on your home screen
  2. Select "Widgets"
  3. Find and add "Termux Widget"
  4. Tap the widget and select "VidChatBox"
üí° The icon will automatically start the server and open the app in your browser.

Step 4: Access the App

You can access VidChatBox in three ways:

  1. Tap the home screen icon (recommended)
  2. Open in browser: http://localhost:3005
  3. Use command line in Termux:
# Start the server vidchatbox start # Check server status vidchatbox status # Stop the server vidchatbox stop # Restart the server vidchatbox restart
